realizar estos pasos en MySQL
    1.Ejecutar el siguiente comando SQL para verificar el modo de autenticación actual: SELECT @@default_authentication_plugin;

    2.Si el resultado muestra 'caching_sha2_password', cambia el modo de autenticación a 'mysql_native_password' ejecutando el siguiente comando SQL:

    3.ALTER USER 'usuario'@'localhost' IDENTIFIED WITH mysql_native_password BY 'contraseña';

    4.Reemplaza "usuario" y "contraseña" con sus propias credenciales de MySQL.

    5.Verifica que el modo de autenticación se ha cambiado correctamente ejecutando el siguiente comando SQL:SELECT @@default_authentication_plugin;
